# Online CV - HTML5 & CSS3 Exercise

A responsive online CV / portfolio page built with semantic HTML5 and modern CSS3.

## Preview

A single-page CV featuring:

- **Header** with navigation, profile photo, name, and tagline
- **About** section with personal info
- **Skills** grid (Front-End, Back-End, Tools)
- **Experience** timeline
- **Projects** showcase with image cards
- **Contact** section with links
- **Footer** with copyright

## Files

| File | Description |
|------|-------------|
| `index.html` | Main CV page |
| `style.css` | Stylesheet with responsive layout |
| `images/` | Profile photo and project screenshots |

## Technologies

- HTML5 semantic elements (`header`, `nav`, `main`, `section`, `article`, `footer`)
- CSS3 Grid, Flexbox, transitions, and media queries
- Google Fonts (Inter, Kaushan Script)

## Getting Started

Open `index.html` in any modern browser. No build tools required.
